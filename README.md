# 💰 Expense Tracker

A full-stack Expense Tracker web app to manage income and expenses with real-time updates and data visualization.

---

## 🚀 Features

- Add income and expenses
- Delete transactions
- Real-time balance calculation
- Category-wise expense tracking
- Interactive pie chart visualization
- Clean dashboard UI

---

## 🛠 Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript
- Chart.js

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

---

### Backend

node server.js
